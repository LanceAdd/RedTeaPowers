# Synthesis And Handoffs

Use this reference when the collection pass is complete enough that the next question is "what now?"

## Evidence Labels

Mark findings clearly:

- `Confirmed fact`: directly supported by source evidence
- `Inference`: likely conclusion drawn from several facts
- `Unknown`: needs another source or explicit confirmation
- `Conflict`: sources disagree or reality is inconsistent

This keeps research from quietly turning into speculation.

When multiple source families are involved, keep the source label beside the finding as well, such as `project-fact`, `official-external`, or `discussion-note`.

## Stop Rules

Stop the research pass when one of these is true:

- the delivery route is now clear
- the main comparison is already decisive
- the remaining unknowns are edge details, not route-changing facts
- more collection would mostly restate what is already known

Escalate instead of researching forever when:

- new evidence keeps expanding the scope boundary
- sources are too contradictory to resolve alone
- the next step requires a product or architecture decision, not more collection

If local facts and external sources disagree and the disagreement changes the route, stop the collection pass and make the conflict explicit before moving on.

## Handoff Guidance

Use:

- `shaping-work` when research changes the route or batch size
- `brainstorming` when facts are gathered but the design choice is still open
- `managing-project-docs` when findings need a durable home
- `writing-plans` when the research result now supports execution planning

## Good Final Section

End a substantial research artifact with:

1. what is now known
2. what is still unknown
3. what should happen next
4. which document or skill should carry the work forward
